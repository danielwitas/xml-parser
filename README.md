# xml-parser 

## clone repository
```
git clone git@github.com:danielwitas/xml-parser.git
cd xml-parser/
```
## Install dependencies
```
composer install
```
## Run tests
```
./vendor/bin/phpunit
```
## Usage
```
copy xml to xml-parser/feeds/
run:
./public/index.php <filename>
```
## Troubleshooting
```
chmod +x public/index.php
```
